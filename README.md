Getting started
1. Set up virtual environment (below is for Mac)
    ```
    python3 -m venv .venv
    ```
2. Install dependant libraries 
    ```
    pip install streamlit openai twilio python-dotenv
    ```
3. Run main.py with Streamlit
   ```
   streamlit run main.py
   ```
